# 🏦 Bank Management System in C

## 📌 Overview
This is a simple Bank Management System developed using C programming.  
It allows users to create accounts, deposit and withdraw money, and check account balance using file handling.

## 🚀 Features
- Create a new bank account  
- Deposit money  
- Withdraw money  
- Check account details and balance  
- Data stored using file handling  

## 🛠️ Technologies Used
- C Programming  
- File Handling (`fopen`, `fread`, `fwrite`)  
- Structures  

## ▶️ How to Run

### Using GCC:
```bash
gcc bank.c -o bank
./bank
